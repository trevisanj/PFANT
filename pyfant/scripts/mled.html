mled - Molecular lines editor

This program allows you to visualize and edit the contents of a molecular lines file such as 

moleculagrade.dat
or 
mghc2cnr_cn3ch13chtiog3tio.dat
.

Opening files

To open a file, invoke the program as follows:

moled.py <filename>

Structure of the program

The graphical interface is organized in a manner that reproduces the hierarchical structure of the molecular lines file, i.e.

  • File properties
  • Molecule #1 properties
    • Set-of-lines #1 (transition) properties
    • Set-of-lines #2 (transition) properties
    • ...s
  • Molecule #2 properties ...
  • ...

Navigation is facilitated by keyboard shortcuts that are shown explicitly.

Editing information

Editable items can be selected using these shortcuts:

  1. Molecule properties: Alt+1
  2. Set-of-lines properties: Alt+2
  3. Molecular lines: Alt+3

Editing (1) or (2) above

Press Enter or double-click with the mouse

Editing molecular lines

Pressing Alt+3 (or its corresponding button) will open another window contaning a table. As you navigate through this table, the current molecular line will be marked in the plots with a big red X.

You can also click with the mouse directly on the plots, causing the nearest wavelength be selected on the table.

Saving the file

Careful: if you press Ctrl+S, the original file will be overwritten with your changes. It is recommended to backup your file before opening it with mled.

To save it with a different name, press Ctrl+Alt+S.

If the file has been changed, but not saved yet, the word "changed" will be shown in the title.
