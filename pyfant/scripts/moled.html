"moled" - Molecular lines editor

This program allows you to visualize and edit the contents of a molecular lines file such as 

moleculagrade.dat
or 
mghc2cnr_cn3ch13chtiog3tio.dat
.

Opening files

To open a file, you have to specify its name from the command line, i.e.,

moled.py <filename>

Structure of the program

The graphical interface is organized in a manner that reproduces the hierarchical structure of the molecular lines file, i.e.

  • File properties
    • Molecule properties
      • Set-of-lines (transition) properties

Navigation is facilitated by keyboard shortcuts that are shown explicitly.

Editing information

Editable items can be selected using these shortcuts:

  1. Molecule properties: Alt+1
  2. Set-of-lines properties: Alt+2
  3. Molecular lines: Alt+3

Editing (1) or (2) above

Press Enter or double-click with the mouse

Editing molecular lines

(3) above will open another window contaning a table. As you navigate through this table, the graphical point corresponding with the current row will be shown in the plots with a big red X.

You can also click with the mouse directly on the plots and the nearest wavelength will be selected on the table.

Saving the file

Careful: if you press Ctrl+S, the original file will be overwritten with your changes. It is recommended to backup your file before opening it with moled.

To save it with a different name, press Ctrl+Alt+S.

If the file has been changed, but not saved yet, the word "changed" will be shown in the title.
