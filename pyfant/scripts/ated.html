ated.py - Atomic lines editor

This program allows you to visualize and edit the contents of a atomic lines file such as 

atomgrade.dat
or 
atom4070g.dat
.

Opening files

To open a file, invoke the program as follows:

ated.py <filename>

Structure of the program

The graphical interface shows the (element name)+(ionization) level on the left, and corresponding plots on the right.

Navigation is facilitated by keyboard shortcuts that are shown explicitly.

Editing atomic lines

Pressing Ctrl+3 (or its corresponding button) will open another window contaning a table. As you navigate through this table, the current atomic line will be marked in the plots with a big red X.

You can also click with the mouse directly on the plots, causing the nearest wavelength be selected on the table.

Saving the file

Careful: if you press Ctrl+S, the original file will be overwritten with your changes. It is recommended to backup your file before opening it with ated.py.

To save it with a different name, press Ctrl+Alt+S.

If the file has been changed, but not saved yet, the word "changed" will be shown in the title.
