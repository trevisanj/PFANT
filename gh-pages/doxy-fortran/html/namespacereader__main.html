subroutine reader_main::assure_read_main ( )

Makes sure that read_main() has been called.

Note
Calls read_main() with argument flag_care_about_dissoc set to .false.
Calls read_main() with argument flag_read_filetoh set to .false.

This routine is used by hydro2, innewmarcs, nulbad. None of these care about dissoc or filetoh variables.
