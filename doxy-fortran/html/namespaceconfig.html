integer config::config_ptdisk = -1

option: –ptdisk

This is about the "number of points" for subroutine fluxis(). Feeds into variable x _ptdisk, which is actually logical. But I needed to make it integer here because I need a tristate variable to flag whether it is kept unitialized (-1).

  • -1 unitialized (hydro2_init() will use main_ptdisk instead)
  • 0 false
  • 1 true
