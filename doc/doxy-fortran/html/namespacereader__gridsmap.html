subroutine reader_gridsmap::read_gridsmap ( )

Reads map of models: dfile:gridsmap.

The information it contains is just a list of .mod files. Files needn't be sorted. Metallicities are taken from the first record of each file and the files are sorted in ascending order of metallicity

This this is necessary because Fortran doesn't have a cross-platform solution for listing the files in a directory.
