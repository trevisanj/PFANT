Working with the GitHub repository

Cloning the repository

Most cases

git clone https://github.com/trevisanj/pfant

Most cases

The repository has two branches:

  • master All Python and Fortran source code
  • gh-pages PFANT website, including Doxygen-generated documentation

When working on your computer locally, in order to have Doxygen parsing files from the master branch while generating output directly into the gh-pages branch, it is necessary to work with the two branches of the repository as parallel local directories. This is achieved by invoking git clone –single-branch option twice.

Here is a set of commands to fetch the two branches separately:

# It is good to be inside a directory called "github" on your computer
$ pwd
/home/j/Documents/projects/pfant/github
# This will clone the "master" branch into a directory called "master"
$ git clone https://github.com/trevisanj/pfant --branch master --single-branch master
# This will clone the "gh-pages" branch into a directory called "gh-pages"
$ git clone https://github.com/trevisanj/pfant --branch gh-pages --single-branch gh-pages
$ ls
gh-pages master

Communicating with GitHub via SSH

To keep git from asking for your github username and password every time you execute a git push, you have to configure git to use the SSH protocol to communicate with github.

The steps involved are roughly summarized below. For full details, please follow the tutorials referenced below.

  1. Generate a SSH key. I followed the tutorial at https://help.github.com/articles/generating-ssh-keys/, but the commands are roughly as follows:

    $ ls -al ~/.ssh # Check existing SSH keys
    $ sh-keygen -t rsa -C "your@email.com"
    $ ssh-add ~/.ssh/id_rsa
    $ pluma ~/.ssh/id_rsa.pub

  2. Copy-paste the key to a box at a settings page at GitHub. After you do this, you can test if it works:

    $ ssh -T git@github.com

  3. Change the remote URL that git uses. Tutorial at https://help.github.com/articles/changing-a-remote-s-url/
    $ git remote set-url origin git@github.com:trevisanj/pfant.git
